# COMM7350 Course Requirement Mapping

This project is designed for the COMM7350 AI and Digital Media Workshop course project.

## Required by the Course

- Python programming project using AI for digital media.
- Topic falls under computer vision.
- Specific purpose and method.
- Findings and results.
- Report around 3 pages, maximum 4 pages, PDF format.
- Report includes objectives, proposed methods, findings/results, conclusion/discussion, and each member's contribution.
- Presentation includes topic/team members, objectives/methods, key findings, conclusion/discussion.
- Everyone in the group speaks.
- Speaker name appears on the slides they present.
- Presentation length: 10 minutes plus 5 minutes Q&A.

## How FaceMood Satisfies This

- Area: computer vision, real-time face analysis, image classification.
- Objective: webcam-based multi-face facial expression recognition.
- Method: OpenCV face detection, MediaPipe landmarks, face crop/alignment, CNN trained on FER2013.
- Findings/results: dataset summary, model accuracy, confusion matrix, screenshots, and demo video.
- Discussion: class imbalance, webcam lighting, face pose, model limitations, future improvements.

## Remaining Evidence to Add

- Trained model checkpoint.
- Training curve.
- Validation/test accuracy.
- Confusion matrix.
- Live demo screenshots.
- Backup demo video.
- Final member contribution table.
